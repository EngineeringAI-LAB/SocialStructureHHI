Abstract

Generating realistic two-person interactions — handshakes, hugs, hand-overs, fights — requires modeling not just each body but the social structure between them: who acts, who reacts, and how contact is established and released over time. We present a phase-decomposed, Ping-Pong approach. A language planner turns a high-level interaction goal into a sequence of structured phases, each with a concrete per-actor description and a duration. An executor diffusion transformer, built on the HunyuanMotion backbone, generates each phase in an ego-centric frame: person 1 acts first (Ping), then person 2 reacts conditioned on person 1's freshly generated motion (Pong). A pinned self-history prefix and per-phase ego↔world tracking remove train/inference mismatch and keep the two trajectories spatially coherent across phases. The released model is trained on Inter-X and InterHuman and supports a full text→motion pipeline.

Method

Three layers: a planner that decomposes the goal, an executor that generates each phase, and an ego↔world state tracker that composes the episode.

global goal ──► LLM planner ──► per-phase P1 / P2 text + duration bucket │ ▼ online Qwen / CLIP text encoding │ ▼ executor (HunyuanMotion-MMDiT + partner conditioning) P1 (Ping): self-history + partner = P2's previous phase P2 (Pong): self-history + partner = P1's just-generated phase │ ▼ ego ↔ world tracking ──► two-person world motion (+ viewer)

Planner

Decomposes a goal into ordered phases (action, body parts, contact, duration), one at a time in a receding-horizon loop.

Executor

HunyuanMotion-Lite backbone with variable-length partner conditioning and a pinned self-history prefix. 201-dim motion (6D rotations + root + FK).

State tracking

Per-phase ego↔world transforms, ground contact, and Ping-Pong overlap stitch phases into one coherent episode.
